5.2.3 Scenario B: Solar Retrofit

üîß

Existing Solar Assessment

Adding battery to an existing solar asset

Key questions: - When does my current PPA expire? - What is the merchant value of my solar asset without battery? - Can I re-contract with battery addition? - What battery return is acceptable to preserve overall project value?

Comparison basis: Co-located BESS vs. Merchant solar exposure (do nothing)

Decision driver: Does battery addition preserve/enhance solar asset value?

‚ö†Ô∏è Retrofit ‚â† Greenfield Economics

In retrofit scenarios, the battery doesn‚Äôt need to compete with standalone returns. It needs to be accretive to the combined asset. A 10% battery IRR may be acceptable if it preserves a solar asset that would otherwise face 5% merchant returns.

5.5.3 Opex Synergies

Co-location also provides ongoing operational savings:

  • O&M: Single site contract (2-5% saving)
  • Insurance: Combined policy (5-10% saving)
  • Land lease: Marginal cost vs. new site
  • Network charges: Potential TUOS/DUOS savings

5.6 Step 5: Assess Contracting Options

The value of co-location often depends on contracting requirements.

5.6.1 Green Premium Assessment

‚ÑπÔ∏è Is a Green Premium Available?

DC-coupled batteries can only charge from solar, creating a ‚Äúguaranteed green‚Äù product. This may command a premium from ESG-focused offtakers.

  • If premium available: Add $5-15/MWh to co-located revenue assumptions
  • If no premium: No adjustment needed

5.6.2 CIS Eligibility

For Australian projects, Capacity Investment Scheme eligibility can be decisive:

Project Type CIS Eligible? Revenue Impact
Standalone BESS Sometimes Depends on tender
Co-located solar+storage Yes Long-term floor price
CIS Value
Government-backed revenue floor reduces financing cost by 50-100bps

If CIS eligibility is critical for your financing structure, this may override location considerations.

5.7.2 Decision Rules

Based on the analysis framework:

Choose Standalone When:
Location value exceeds infrastructure synergies

MLF revenue gain > Capex synergy saving, AND you can accept higher merchant exposure, AND CIS eligibility is not critical.

Choose Co-located When:
Infrastructure synergies and contracting value exceed location penalty

Capex synergy + Green premium + CIS value > MLF penalty, OR this is a retrofit scenario where battery preserves solar asset value.

5.9.2 Final Decision Framework

‚úÖ

Decision Summary

Synthesizing the analysis

If greenfield and location-flexible: - Default to standalone unless co-location provides clear IRR advantage - Location value typically exceeds infrastructure synergies

If greenfield but CIS-dependent: - Co-location likely preferred for eligibility - Accept some location penalty for financing benefit

If retrofit with expiring PPA: - Co-location typically preferred - Battery protects solar asset value - Lower battery IRR acceptable if combined returns meet hurdle

If retrofit but solar has strong MLF: - Evaluate standalone at same site (AC-coupled, separate DUID) - May capture both location value and synergies

üö® Remember

This framework provides structure, but every project is unique. The quantitative analysis should inform‚Äînot replace‚Äîcommercial judgment. Engage technical and financial advisors to validate assumptions before committing capital.

Key Takeaways
  • First classify your scenario: greenfield vs. retrofit changes the decision framework
  • Establish a clear return benchmark before evaluating options
  • Quantify location value (MLF impact) and infrastructure synergies separately
  • Consider contracting requirements: green premium, CIS eligibility, PPA structure
  • Use dispatch modeling to validate assumptions and test sensitivities
  • For greenfield: standalone typically wins unless specific co-location benefits apply
  • For retrofit: co-location typically wins as it protects existing solar value
  • Every project is unique‚Äîuse this framework as a guide, not a prescription

- Accept some location penalty for financing benefit

**If retrofit with expiring PPA:**
- Co-location typically preferred
- Battery protects solar asset value
- Lower battery IRR acceptable if combined returns meet hurdle

**If retrofit but solar has strong MLF:**
- Evaluate standalone at same site (AC-coupled, separate DUID)
- May capture both location value and synergies
